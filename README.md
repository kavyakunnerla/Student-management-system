# Student-management-system
This is a **web-based Student Management System** built using **HTML, CSS, and JavaScript**.  
The application allows users to **login, add student details, display a dynamic list of students, and delete records**.

## Features
- Simple **login system** for admin access
- Add students with **Name, Age, and Class**
- Display all students in a **dynamic table**
- Delete student records with a click of a button
- Interactive and user-friendly **frontend UI**
- Demonstrates **DOM manipulation and client-side programming**

## Skills Demonstrated
- HTML, CSS, JavaScript
- DOM manipulation and event handling
- Dynamic table creation and updates
- Basic login authentication
- Frontend development and user interaction

## How to Use
1. Open `index.html` in a web browser.
2. Login with:
   - Username: `admin`
   - Password: `1234`
3. Add student details in the form and click **Add Student**.
4. View all students in the table and delete if needed.

## Future Enhancements
- Add **localStorage or database** support to store data permanently.
- Add **edit/update student records** feature.
- Improve UI with frameworks like **Bootstrap** or **Tailwind CSS**.
- Add **search and filter functionality** for students.
